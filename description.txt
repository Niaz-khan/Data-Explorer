Simple Data Explorer

A GUI data science project using Python and Tkinter. Upload a CSV file, view summary statistics, and visualize the data using histograms, scatter plots, and bar charts. A simple tool for exploratory data analysis.

Features:

- File uploader
- Summary statistics
- Plot options (histogram, scatter plot, bar chart)
- Interactive GUI

Technologies:

- Python
- Tkinter
- Pandas
- Matplotlib

Usage:

1. Upload a CSV file
2. Select plot options
3. Click "Plot Data" to visualize the data
